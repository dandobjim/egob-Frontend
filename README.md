This is a [React](https://es.reactjs.org/) project bootstrapped with [`react-bootstrap`](https://react-bootstrap.github.io/).

## Installing packages
Before all, install the necessaries packages
```bash
sudo apt install nodejs
sudo apt install npm
npm i
```


## Start
Run the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
